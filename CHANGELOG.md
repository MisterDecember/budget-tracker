# Changelog

All notable changes to PixelVault will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-01-18

### Added
- Initial release
- User authentication with local storage
- Account management (Checking, Savings, Credit, Cash, Investment)
- Transaction tracking with categories
- Debt tracking with amortization calculations
- Recurring transaction management
- Financial forecasting with charts
- Dashboard with net worth overview
- Dark retro-inspired theme with neon accents

### Technical
- React 18 + TypeScript
- Vite 5 build system
- Tailwind CSS styling
- shadcn/ui component library
- Zustand state management
- IndexedDB for local data persistence
- Recharts for data visualization
